# Package index

## All functions

- [`abronia`](abronia.md) : Occurrence records of Abronia graminea

- [`bg_by_date()`](bg_by_date.md) : Function to obtain environmental
  background organized by date

- [`cells2samp()`](cells2samp.md) : Helper function to randomly select
  cell IDs for generating environmental background data.

- [`clean_dup()`](clean_dup.md) : Function to thin longitude and
  latitude data

- [`clean_dup_by_date()`](clean_dup_by_date.md) : Function to thin
  occurrence data Cleans up duplicated longitude and latitude data by
  year using a specified distance threshold. The distance can be
  specified as a geographic distance or, if a raster_mask is provided,
  as a pixel distance.

- [`colors`](colors.md) : Colors for plotting

- [`correlation_finder()`](correlation_finder.md) : Function to find
  strong correlations within environmental predictors

- [`cov_center()`](cov_center.md) : Function to compute the covariance
  matrix of an ellipsoid niche model.

- [`ellipsoid_omr()`](ellipsoid_omr.md) : Compute omission rate and
  statistical metrics for ellipsoid models.

- [`ellipsoid_projection()`](ellipsoid_projection.md) :
  ellipsoid_projection: function to project an ellipsoid model

- [`ellipsoid_selection()`](ellipsoid_selection.md) :
  ellipsoid_selection: Performs models selection for ellipsoid models

- [`ex_by_date()`](ex_by_date.md) : Extract environmental data by date

- [`inEllipsoid()`](inEllipsoid.md) : inEllipsoid: Determine if a point
  is inside or outside an ellipsoid

- [`metaras()`](metaras.md) : Helper function to obtain layer name from
  a raster layer

- [`pROC()`](pROC.md) : Partial ROC calculation for Niche Models

- [`plot_ellipsoid()`](plot_ellipsoid.md) : Function to plot ellipsoid
  models in E-space

- [`predict(`*`<sp.temporal.selection>`*`)`](predict.md) : Predict the

- [`sp.temporal.bg-class`](sp.temporal.bg-class.md) :

- [`sp.temporal.env-class`](sp.temporal.env-class.md) :

- [`sp.temporal.modeling-class`](sp.temporal.modeling-class.md) :

  S3 classes to organize data and results of `tenm` objects

- [`sp.temporal.selection-class`](sp.temporal.selection-class.md) :

  S3 classes to organize data and results of `tenm` objects

- [`sp_temporal_data()`](sp_temporal_data.md) : Function to create a
  Species Temporal Data object (STD object).

- [`tdf2swd()`](tdf2swd.md) : Temporal data.frame to Samples With Data
